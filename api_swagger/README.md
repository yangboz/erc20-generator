# CRUD-NodeJS-Swagger-MongoDB

A simples CRUD example by using NodeJS, Swagger UI and MongoDB.

## Technologies:

- NodeJS Express
- Swagger UI
- MongoDB

## To run this project

- Install packages:
  `npm install`
- Run API:
  `nodemon server`
- Acess URL:
  `http://localhost:3000`
